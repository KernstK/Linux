Команды для терминала Linux

  1. ls - проверка файлов в определенном местоположении компьютера
  2. mkdir - создание папки
  3. touch - создание текстового файла
  4. cd/cd ../ - открытие какой-нибудь папки или файла/выход из папки
  5. code - изменить код чего-то
  6. rm/rmdir - удаление файла/удаление директории
